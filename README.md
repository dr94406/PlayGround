# PlayGround
최대한 작게 시작하여 자주 실패하는 것을 목표로 하는 <br>
내 마음과 내 뜻대로 만들어 가는 저장소  플레이 그라운드!

기술

- Java 11
- Spring Boot 2.7
- MyBatis
- Gradle
- Spring Security (ver.2)
- JWT(ver.2) <br>
